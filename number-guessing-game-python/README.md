# Number Guessing Game (Python)

A small interactive Python game where the computer randomly selects a number between 1 and 100.  
The player has to guess the correct number with hints from the program.

## Features
- Random number generation
- User input
- Feedback if the guess is too high or too low
- Loop until the correct number is guessed

## Technologies Used
- Python
- random module

## Example

Welcome to the Number Guessing Game!
I am thinking of a number between 1 and 100.

Enter your guess: 40
Too low ⬇️ Try again.

Enter your guess: 80
Too high ⬆️ Try again.

Enter your guess: 63
🎉 Correct! The number was 63.

This project was created to practice:
